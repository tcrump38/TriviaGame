# TriviaGame

Link: https://tcrump38.github.io/TriviaGame/

This is a timed, multiple choice trivia game based on The Lord of the Rings Trilogy. I had to discover how to user timers and pull information from arrays.

Users will initiate the moment.js, timed scripted quiz:

Users have a 15 second countdown per trivia question:
<img width="1279" alt="Screen Shot 2019-03-15 at 1 27 19 PM" src="https://user-images.githubusercontent.com/39817046/54454068-a1c6cb00-4726-11e9-8430-fb599d088b63.png">

Results are served relaying correct, incorrect, and unanswered questions. Users can restart the quiz:
